A simple, interactive Tic Tac Toe game built with HTML, CSS, and JavaScript. Play with a friend, enjoy sound effects, and experience a smooth, animated interface inspired by classic pen-and-paper fun!

![Dancing Cat Winner Animation](./excited.gif)

---

## Features

- **Classic 3x3 Grid**: Play as "X" or "O" in a traditional Tic Tac Toe setup.
- **Interactive UI**: Click boxes to make your move; current turn is always displayed.
- **Winning Animation**: On win, a celebratory dancing cat GIF (see above) appears.
- **Sound Effects**: Fun sounds for turns, victory, and game over (make sure `music.mp3`, `ting.mp3`, and `gameover.mp3` are included).
- **Responsive Design**: Looks great on desktop and mobile.
- **Easy Reset**: Click "Reset" to start a new game at any time.

---

## How to Run

1. **Clone or Download the Repo**.
2. Make sure these files are present in the project folder:
    - `index.html`
    - `style.css`
    - `app.js`
    - `excited.gif` (the winner animation)
    - `music.mp3`, `ting.mp3`, `gameover.mp3` (sound effects)
3. Open `index.html` in your browser.
4. Play and have fun!

---

## File Overview

- **index.html**: Game board, info display, and layout.
- **style.css**: Styling for the board, animation, and responsive design.
- **app.js**: Game logic, win detection, sound/animation control.
- **excited.gif**: (as shown above) Animated cat that celebrates the winner.
